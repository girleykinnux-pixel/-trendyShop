# TrendyShop - Tienda de Moda

TrendyShop es una aplicación web interactiva y minimalista diseñada para la gestión de un catálogo de ropa y maquillaje y un sistema de carrito de compras en tiempo real. Este proyecto fue desarrollado en colaboración utilizando Git y GitHub para la integración de módulos de desarrollo.

---

## Características Principales
* **Diseño Minimalista Premium:** Interfaz limpia basada en una paleta de colores corporativa (Verdes y tonos claros).
* **Navegación Fluida:** Barra de menú unificada y adaptable en todas las secciones.
* **Carrito de Compras Reactivo:** Añade productos y visualiza la actualización del contador en tiempo real.
* **Formulario de Contacto Estilizado:** Sección dedicada para la atención al cliente y soporte.

---

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica de las páginas.
* **CSS3 (Custom Properties):** Estilos personalizados y diseño responsivo.
* **Bootstrap 5:** Sistema de rejilla (Grid) y componentes de interfaz premium.
* **JavaScript (Vanilla):** Lógica del carrito de compras y persistencia de datos.
